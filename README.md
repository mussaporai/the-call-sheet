# THE CALL SHEET

Dashboard de gestão de mega eventos corporativos — do briefing ao wrap.

Single-page app (HTML + CSS + Vanilla JS) com login, checklists de Pré-Evento / Dia D / Pós-Evento, fornecedores, budget, briefings, reuniões com resumo via IA e cronograma. Estado persistido em `localStorage`.

## Uso

Abra `index.html` em qualquer navegador, ou publique via GitHub Pages.

**Credenciais de acesso:**
- `admin` / `sfera2026` (acesso total)
- `equipe` / `evento2026` (acesso operacional)
- `cliente` / `view2026` (somente leitura — visual, sem restrição funcional ainda)
